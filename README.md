This project performs Exploratory Data Analysis (EDA) on a customer sales dataset using Python, Pandas, Matplotlib, and Seaborn. The analysis involved data cleaning, preprocessing, and visualization to understand purchasing patterns across different customer segments such as states, gender, age groups, and occupations. The project identifies potential customer segments and top-selling product categories, providing insights that can help businesses improve customer targeting, inventory planning, and overall sales strategies. Overall, the project demonstrates practical skills in data analysis, data manipulation, and visualization to extract meaningful business insights from raw data.
